# Skills.md and Agents.md

Skills.md and Agents.md are emerging open formats for providing structured guidance to AI agents —
defining what an agent can do, how it should behave, and what skills it has available.

## The Idea

Instead of embedding all agent instructions in code, these files provide a declarative, human-readable
way to configure agent behavior. Similar to how CLAUDE.md guides Claude Code, these formats aim
to standardize how agents receive their instructions.

## Skills.md

Defines the capabilities (skills) available to an agent — what they do, when to use them,
and how they're invoked.

## Agents.md

Defines an agent's identity, behavior guidelines, constraints, and goals in a structured format
that the agent reads and follows.
