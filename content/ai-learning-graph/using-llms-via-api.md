# Using LLMs via API

Most LLM providers expose their models through REST APIs. Understanding the API layer is essential
for building applications that integrate language models programmatically.

## Key Topics

- **[[message-roles]]** — how conversations are structured with system, user, and assistant messages
- **[[llm-parameters]]** — controlling output behavior with temperature, top-p, and other settings
- **[[llm-tools-and-functions]]** — giving models the ability to call external tools and functions
- **[[openai-responses-api]]** — OpenAI's Responses API and how it works
