# Machine Learning

Machine Learning is a subfield of AI where systems learn patterns from data rather than following explicitly programmed rules. A model improves its performance on a task through exposure to examples.

## Core Paradigms

- **[[supervised-learning]]** — learning from labeled input-output pairs
- **[[unsupervised-learning]]** — finding structure in unlabeled data
- **[[model-evaluation]]** — measuring how well a model actually works

## Key Concepts

**Training** — the process of adjusting model parameters to minimize error on a dataset.

**Generalization** — a model's ability to perform well on data it hasn't seen during training. Overfitting (memorizing training data) is the primary failure mode to avoid.

**Features** — the inputs fed to a model. Feature quality often matters more than model choice.

## Relationship to Deep Learning

Traditional ML uses hand-crafted features. Deep learning (a subset) learns features automatically from raw data using neural networks, which scales better with large datasets.
